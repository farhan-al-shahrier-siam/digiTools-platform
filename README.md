# 🚀 DigiTools Platform

A modern and user-friendly platform that provides premium digital tools to help users boost productivity and create professional outputs effortlessly.

---

## 📖 Description

**DigiTools Platform** is a web-based application where users can explore and access high-quality digital tools such as resume kits and productivity resources. The platform focuses on simplicity, clean UI, and smooth user experience while managing products and interactions efficiently.

---

## ⚙️ Technologies Used

- ⚛️ React.js  
- 🎨 Tailwind CSS & DaisyUI  
- 📜 JavaScript (ES6+)  
- 🔔 React-Toastify (NPM Package)  
- 📦 JSON (for product data)

---

## ✨ Features

- 🛒 **Dynamic Product Listing**  
  Browse digital products loaded from JSON with a clean and responsive UI.

- 🔔 **Interactive Notifications**  
  Real-time feedback using toast messages (add/remove actions, etc.).

- 💡 **Modern UI/UX Design**  
  Built with Tailwind CSS & DaisyUI for a sleek and responsive experience.
